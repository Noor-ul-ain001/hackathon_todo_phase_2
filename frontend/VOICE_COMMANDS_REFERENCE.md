1. **Be Specific**: Use exact task names for best matching
   - âœ… Good: "Complete task buy milk"
   - âŒ Less reliable: "Complete buy"

2. **Update Format**: Always use "to" when updating
   - âœ… Good: "Update task homework to study for exam"
   - âŒ Won't work: "Update task homework study for exam"

3. **Natural Variations**: These work too!
   - "Add task" = "Create task" = "New task"
   - "Complete" = "Mark done" = "Finish" = "Check off"
   - "Delete" = "Remove"

4. **Wait for Feedback**:
   - Green dot = Listening
   - Processing spinner = Executing command
   - Action message = Confirmation

5. **Task Names**: If you have a task named "Buy groceries and milk":
   - Say: "Complete task buy groceries and milk"
   - Partial matches work: "Complete task buy groceries"

## ğŸ” Troubleshooting

**Command not recognized?**
- Use the exact format shown above
- Check the help panel in the voice chat modal
- Speak clearly and wait for the green listening indicator

**Task not found?**
- Use the exact task name
- Try a partial match (system searches for closest match)
- Say "List tasks" to hear all task names

**No response?**
- Check microphone permissions
- Ensure you see the green "Listening" indicator
- Look for error messages in the modal
